Weekly Blog

Week 1: October 5, 2018

This week out goal was to lay the foundation for our project, which involved the following: establishing a website, coming up with a narrative, creating concept sketches and familiarzing ourselves with key features we will need in development. One of these features was the integration of the Virtual Reality Tool Kit (VRTK), which will help us streamline development moving forward. This is in part thanks to the built in prefabs for integration into Unity. Allowing for the Oculus rift to be programmed and demo'd real time, with or without a headset setup. But also cuts the time down in half, making objects interactable etc. thanks to the builder tools provided.
