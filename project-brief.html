Project Brief

Our idea for the final project is a mystery and adventure-themed game that allows the user to control a character in a maze-like forest environment. The game will be based on a narrative that begins with the main character losing his/her pet dog that runs off into the woods. The userâ€™s main task is to traverse the labyrinth and find the missing dog.

We intend to design a compelling and immersive environment that will evoke the intended emotive experience of fear and anxiety. This will be achieved through the modelling and texturing of the landscape, through lighting and shadows and through music and sound effects. The interaction in our game will mainly be through clues that the user will encounter along the way. We also want to achieve some interaction with the environment itself like for example having scenarios where the user is able to pick up and move objects like tree branches that are blocking the paths. We hope to make the game challenging but also manageable in order to keep the user interested and fully immersed.
